
# Introduction

Welcome to the Hammer 5 Tools documentation. Hammer 5 Tools is a powerful, open-source toolkit designed to streamline the Counter-Strike 2 Workshop workflow for mapmakers and technical artists.

![Hero Video](videos/hero.mp4)

> [!NOTE]
> **Source 2 Ready**
> Hammer 5 Tools is built specifically for the Source 2 engine and Counter-Strike 2 Workshop Tools.

> [!TIP]
> This documentation is now built using Markdown, making it incredibly easy to edit and contribute to!

## Key Features
- **SmartProp Editor:** Intuitive interface for creating complex procedural props.
- **SoundEvent Editor:** Real-time editing and previewing of game sounds.
- **Map Builder:** Streamlined compilation and baking processes.
- **Asset Management:** Tools to clean up and organize project files.
- **VSmart Library Integration:** Access a growing collection of community-made SmartProps.
