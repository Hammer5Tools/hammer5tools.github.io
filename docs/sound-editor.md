# Sound Event Editor

The Sound Event Editor allows you to manage in-game sound configurations seamlessly.

## Working with VSND Files
Import your `.vsnd` files directly. You can preview sounds within the editor before adding them to your sound events.

## Presets
Quickly apply standard Source 2 sound event parameters using built-in presets. These includes 3D spatialization, occlusion settings, and random pitch variations.

## Syncing with Addon
All changes are written directly to `soundevents_addon.vsndevts`. Hammer 5 Tools handles the syntax formatting for you.
