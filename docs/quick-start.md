# Quick Start Guide

Get up and running with your first project using Hammer 5 Tools.

## Step 1: Open Your Addon
Launch the tool and select your CS2 addon folder. The tool will automatically scan for relevant files like `soundevents_addon.vsndevts` and `.vsmart` files.

## Step 2: Create a Sound Event
Navigate to the Sound Event Editor tab. Click **"Add New Event"**, select your `.vsnd` file, and choose a preset like `cs_default_3d`. Save the file, and your sound is ready for use in Hammer or game scripts.

## Step 3: Build Your Map
Use the Map Builder to compile your map with custom lightmap resolutions. Monitor the build progress in real-time within the app.
