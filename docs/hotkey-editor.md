# Hotkey Editor

Customize your workshop tools shortcuts to match your workflow and improve productivity.

## Hotkey Configuration
The interface lists all tool-specific actions and their currently assigned keys. Use the **Edit** action to change a binding or **Reset** to return to defaults.

## Managing Layouts
Store multiple shortcut layouts for different mapping tasks. Switch between them instantly without having to re-configure individual keys. 

> [!NOTE]
> Custom keybinds are stored in your user profile, making them persistent across different addons.
