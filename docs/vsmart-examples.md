## Set Variable

Learn how to use modifiers to set boolean values and use them for conditional logic.

![Set Variable Example 1](https://i.giphy.com/media/v1.Y2lkPTc5MGI3NjExMWo3djg4N2V3ejl5azR3aGl2ZzhvazB2Z3Y2YWxvbGFzcWZjMjVidyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/Ol6CWd48l5Vx4b4PVn/giphy.gif)

![Set Variable Example 2](https://i.giphy.com/media/v1.Y2lkPTc5MGI3NjExNXozd3pzeTcwa3BtcHl0YjF2OWxlMDE1MzN3NDIzcXY5ZnUzY2VxMSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xFz00s7CSm6C1h44xM/giphy.gif)

[Download variable_set_example.vsmart](static/vsmart-examples/variable_set_example.vsmart)

> [!NOTE]
> **Modifier Ordering**
> Modifier ordering is critical in this case. You must first set the boolean value before you can use it in subsequent elements or modifiers.
