# Smart Props Editor

Edit `.vsmart` files using an intuitive node-like property interface. Create complex procedural models with ease.

## Real-time Preview
While editing properties, the tool provides feedback on how your prop will be constructed. Adjust position, rotation, and scaling variables on the fly.

## Compatibility
Maintains compatibility with all standard Source 2 SmartProp features, allowing you to use your props in any CS2 map.
