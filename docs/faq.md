# FAQ & Troubleshooting

### The tool won't launch!
Ensure you have the latest .NET Desktop Runtime installed.

---



### My VSmart files fail to compile in the editor, how to fix?
To compile VSmart files, launch the Hammer Editor through Hammer 5 Tools.

---


### There is no sound in hammer editor, what's wrong?
You need to create an event for each sound that you have imported. If you do, but the sound is still missing in the soundevent picker, you need to run the map in the editor: F9 > Run (Skip build).

> [!WARNING] 
The map you ran must be in the maps folder, and it's recommended to keep the name the same as the addon.
