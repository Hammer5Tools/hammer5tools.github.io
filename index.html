Project Tools

SoundEvent Editor

Manage in-game sounds with ease. This editor works with Counter-Strike 2 sounds (vsnd files) and lets you explore, preview, and set up sounds in a sound event. Use presets to quickly add sound events, or create your own presets. This tool edits the soundevents_addon.vsndevts file.

SmartProp Editor

Edit Vsmart files with an easy interface. This tool supports many properties and preserves compatibility with older formats. You can edit properties in real-time, usefully to adjust position or rotation of models. Use presets to quickly create repeatedly elements.

Loading Editor

Customize loading screens with images, descriptions, and icons. Make them unique and appealing for community maps. You can preview screenshots captured with the "png_screenshot" console command or add your own images.

Hotkey Editor

Set and manage keyboard shortcuts for many tasks. This tool can simplify your workflow by making important functions easy to reach. You can store several hotkey presets for different projects and switch between them as needed.

AssetGroup Maker

Create many assets at once for Source 2. You only need a simple config file and a folder scan to generate files for large asset sets. This system keeps everything consistent across your assets and saves time by avoiding manual edits.
